## Commit Message Patterns

### Feature Development
- "feat: add [component/feature] for [purpose]"
- "feat: implement [functionality] with [details]"
- "feat: create [system] to handle [use case]"

### Bug Fixes
- "fix: resolve [issue] in [component]"
- "fix: correct [problem] affecting [functionality]"
- "fix: address [bug] causing [symptom]"

### Improvements
- "improve: enhance [component] with [feature]"
- "improve: optimize [system] for [benefit]"
- "improve: refactor [code] for [reason]"

### Documentation
- "docs: update [documentation] for [topic]"
- "docs: add [information] to [section]"
- "docs: improve [guide] with [details]"

### Testing
- "test: add tests for [component/feature]"
- "test: improve test coverage for [module]"
- "test: fix failing tests in [area]"

### Security
- "security: implement [security measure]"
- "security: fix [vulnerability] in [component]"
- "security: enhance [protection] for [system]"

### Performance
- "perf: optimize [component] for [improvement]"
- "perf: improve [system] performance"
- "perf: reduce [resource] usage in [area]"
